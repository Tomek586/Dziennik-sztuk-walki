# Tu wrzuć pliki .fbx z Mixamo

Postać: **X Bot** · Ustawienia pobierania: FBX Binary, **With Skin**, 30 FPS,
Keyframe Reduction: none, „In Place" zaznaczone gdy dostępne.

| Szukaj | Zapisz jako |
|---|---|
| fight idle | `fight-idle.fbx` |
| jab cross | `jab-cross.fbx` |
| hook | `hook.fbx` |
| roundhouse | `roundhouse.fbx` |
| mma kick | `mma-kick.fbx` |
| leg sweep | `leg-sweep.fbx` |
| takedown / shoot / tackle | `takedown.fbx` |
| victory | `victory.fbx` |

Po wrzuceniu plików daj znać — skonwertuję je do `.glb` (folder `../models/`)
i podepnę pod sceny. Bez plików strona pokazuje elegancki fallback (enso z tuszu).
